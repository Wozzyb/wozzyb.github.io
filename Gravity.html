Gravity

Gravity doesn't always need to go down

Role: Level Designer

Role: Game Designer

Game Type: Quarter Game

This was a quick game jam I signed up for during my last winter quarter at UCI. The theme of the game was one tool many used, so we picked a gravity gun.

I was responsible for designing the mechanics, the gun’s uses, as well as user experience. I had fun making the tutorial levels and sketching and prototyping levels for the game. I also had fun designing the uses of the guns as well as coming up with the puzzle systems we’d be using to make the game interesting.

The challenges I faced during making this game though were the language barrier and the time difference between everyone. The team was spread across time zones, and we span multiple cultures. There were days when we’d be on a discord call using Google Translate to tell each other what we were saying, and even with all these obstacles in mind we managed to make a product I'm proud of
